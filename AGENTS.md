# AGENTS.md

## 프로젝트 성격

이 프로젝트는 C++ 초보 학습용 프로젝트입니다. LocalAIChat을 단계적으로 확장하면서 C++ 문법, 프로그램 구조, 자료구조, 메모리, 알고리즘 기초를 익히는 것이 목적입니다.

## 작업 규칙

1. 사용자가 요청한 단계의 범위만 구현합니다.
2. 다음 단계 기능을 미리 구현하지 않습니다.
3. 7단계까지는 `main.cpp` 하나를 중심으로 유지하고, 여러 `.cpp`/`.h` 파일로 분리하지 않습니다.
4. 7단계에서는 기존 기능을 클래스 기반 구조로 리팩토링만 합니다.
5. 새 기능 추가보다 역할 분리와 구조 개선에 집중합니다.
6. 7단계부터 객체지향 리팩토링 목적에 한해 `class` 사용을 허용합니다.
7. 사용할 클래스는 `Message`, `ChatBot`, `HistoryManager`, `CommandHandler`, `ChatApp`으로 제한합니다.
8. 대화 기록 저장과 불러오기 목적에 한해 `std::ifstream`, `std::ofstream`, `chat_history.txt`를 사용합니다.
9. 5단계의 규칙 기반 응답 알고리즘과 4단계 명령어 시스템을 유지합니다.
10. 명령어는 `/help`, `/history`, `/clear`, `/exit`로 유지합니다.
11. 일반 대화 응답은 `ChatBot`에서 `std::string::find` 기반 키워드 검색으로 처리합니다.
12. 대화 기록은 `Message`와 `std::vector<Message>`를 사용합니다.
13. 사용자가 명시적으로 요청하기 전까지 `map`, 포인터, 동적 메모리 할당을 사용하지 않습니다.
14. 실제 AI 모델 연결은 후반 단계까지 구현하지 않습니다.
15. Ollama, OpenAI API, HTTP 요청, 서버 기능은 후반 단계까지 구현하지 않습니다.
16. 상속, 다형성, 템플릿, 람다식, 스마트 포인터를 사용하지 않습니다.
17. GUI로 전환하지 않습니다.
18. CMake 프로젝트로 전환하지 않습니다.
19. 코드 설명은 C++ 초보자 기준으로 작성합니다.
20. Visual Studio 2022와 Windows 환경을 우선 기준으로 합니다.
21. 변경 후에는 가능한 범위에서 빌드 검증을 수행합니다.
22. 빌드 검증이 성공하면 로컬 git commit을 생성할 수 있습니다.
23. git push, git remote add, GitHub 연결은 절대 하지 않습니다.
