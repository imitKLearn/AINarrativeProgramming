# AGENTS.md

## 프로젝트 성격

이 프로젝트는 C++ 초보 학습용 프로젝트입니다. LocalAIChat을 단계적으로 확장하면서 C++ 문법, 프로그램 구조, 자료구조, 메모리, 알고리즘 기초를 익히는 것이 목적입니다.

## 작업 규칙

1. 사용자가 요청한 단계의 범위만 구현합니다.
2. 다음 단계 기능을 미리 구현하지 않습니다.
3. 6단계까지는 `main.cpp` 하나를 중심으로 유지하고, 여러 `.cpp`/`.h` 파일로 분리하지 않습니다.
4. 6단계에서는 대화 기록 저장과 불러오기 목적에 한해 파일 입출력을 사용합니다.
5. 6단계 파일 입출력은 `std::ifstream`, `std::ofstream`, `chat_history.txt` 사용으로 제한합니다.
6. 5단계의 규칙 기반 응답 알고리즘과 4단계 명령어 시스템을 유지합니다.
7. 4단계에서 구현한 명령어는 `/help`, `/history`, `/clear`, `/exit`로 유지합니다.
8. 명령어 처리는 `handleCommand` 함수에서 담당합니다.
9. 일반 대화 응답은 `getResponse` 함수에서 `std::string::find` 기반 키워드 검색으로 처리합니다.
10. 대화 기록은 `struct Message`와 `std::vector<Message>`를 사용합니다.
11. 사용자가 명시적으로 요청하기 전까지 `class`, `map`, 포인터, 동적 메모리 할당을 사용하지 않습니다.
12. 실제 AI 모델 연결은 후반 단계까지 구현하지 않습니다.
13. Ollama, OpenAI API, HTTP 요청, 서버 기능은 후반 단계까지 구현하지 않습니다.
14. GUI로 전환하지 않습니다.
15. CMake 프로젝트로 전환하지 않습니다.
16. 코드 설명은 C++ 초보자 기준으로 작성합니다.
17. Visual Studio 2022와 Windows 환경을 우선 기준으로 합니다.
18. 변경 후에는 가능한 범위에서 빌드 검증을 수행합니다.
19. 빌드 검증이 성공하면 로컬 git commit을 생성할 수 있습니다.
20. git push, git remote add, GitHub 연결은 절대 하지 않습니다.
