# AGENTS.md

## 프로젝트 성격

이 프로젝트는 C++ 초보 학습용 프로젝트입니다. LocalAIChat을 단계적으로 확장하면서 C++ 문법, 프로그램 구조, 자료구조, 메모리, 알고리즘 기초를 익히는 것이 목적입니다.

## 작업 규칙

1. 사용자가 요청한 단계의 범위만 구현합니다.
2. 다음 단계 기능을 미리 구현하지 않습니다.
3. 9단계까지는 `main.cpp` 하나를 중심으로 유지하고, 여러 `.cpp`/`.h` 파일로 분리하지 않습니다.
4. 9단계에서는 Ollama 로컬 AI 연결만 추가합니다.
5. OpenAI API, 클라우드 API, 원격 서버 연결, API Key 사용은 하지 않습니다.
6. 외부 라이브러리 설치, vcpkg 설치, 환경변수 설정을 요구하지 않습니다.
7. 실제 AI 모델 연결, Ollama, HTTP 요청은 9단계 로컬 AI 연결 목적에 한해 허용합니다.
8. 로컬 AI 연결 실패 시 기존 규칙 기반 응답으로 fallback합니다.
9. 기존 채팅 기능, 명령어, 메모리 학습, 기록 저장 기능은 유지합니다.
10. 명령어는 `/help`, `/history`, `/clear`, `/exit`, `/memory`로 유지합니다.
11. 일반 대화 응답은 `ChatBot`에서 먼저 로컬 AI 응답을 시도하고 실패하면 규칙 기반 응답을 사용합니다.
12. 대화 기록은 `Message`와 `std::vector<Message>`를 사용합니다.
13. 사용자가 명시적으로 요청하기 전까지 `map`을 사용하지 않습니다.
14. 상속, 다형성, 템플릿 직접 작성, 람다식, `shared_ptr`, `weak_ptr`를 사용하지 않습니다.
15. 복잡한 포인터 연산, 배열 포인터, 이중 포인터, `malloc/free`를 사용하지 않습니다.
16. GUI로 전환하지 않습니다.
17. CMake 프로젝트로 전환하지 않습니다.
18. 코드 설명은 C++ 초보자 기준으로 작성합니다.
19. Visual Studio 2022와 Windows 환경을 우선 기준으로 합니다.
20. 변경 후에는 가능한 범위에서 빌드 검증을 수행합니다.
21. 빌드 검증이 성공하면 로컬 git commit을 생성할 수 있습니다.
22. git push, git remote add, GitHub 연결은 절대 하지 않습니다.
