# AGENTS.md

## 프로젝트 성격

이 프로젝트는 C++ 초보 학습용 프로젝트입니다. LocalAIChat을 단계적으로 확장하면서 C++ 문법, 프로그램 구조, 자료구조, 메모리, 알고리즘 기초를 익히는 것이 목적입니다.

## 작업 규칙

1. 사용자가 요청한 단계의 범위만 구현합니다.
2. 다음 단계 기능을 미리 구현하지 않습니다.
3. 4단계까지는 `main.cpp` 하나를 중심으로 유지하고, 여러 `.cpp`/`.h` 파일로 분리하지 않습니다.
4. 4단계에서는 명령어 시스템만 추가합니다.
5. 4단계에서 구현하는 명령어는 `/help`, `/history`, `/clear`, `/exit`로 제한합니다.
6. 명령어 처리는 `handleCommand` 함수로 분리합니다.
7. 대화 기록은 `struct Message`와 `std::vector<Message>`를 사용합니다.
8. 사용자가 명시적으로 요청하기 전까지 `class`, `map`, 파일 입출력, 포인터, 동적 메모리 할당을 사용하지 않습니다.
9. 실제 AI 모델 연결은 후반 단계까지 구현하지 않습니다.
10. Ollama, OpenAI API, HTTP 요청, 서버 기능은 후반 단계까지 구현하지 않습니다.
11. GUI로 전환하지 않습니다.
12. CMake 프로젝트로 전환하지 않습니다.
13. 코드 설명은 C++ 초보자 기준으로 작성합니다.
14. Visual Studio 2022와 Windows 환경을 우선 기준으로 합니다.
15. 변경 후에는 가능한 범위에서 빌드 검증을 수행합니다.
16. 빌드 검증이 성공하면 로컬 git commit을 생성할 수 있습니다.
17. git push, git remote add, GitHub 연결은 절대 하지 않습니다.
18. 모든 작업은 `D:\KM\K82\NA\NA` 폴더 안에서만 수행합니다.
